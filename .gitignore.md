## ğŸ“ **For New Team Members**

When cloning the repo, you'll need to:

1. **Get the data files**
   - Download source TTL files OR
   - Get `deduplicated_knowledge_graph.ttl` from team

2. **Install dependencies**
   ```bash
   # Backend
   cd backend && pip install -r requirements.txt
   
   # Frontend
   cd frontend && npm install
   
   # KG tools
   cd knowledge-graph && pip install -r requirements.txt
   ```

3. **Set up environment**
   ```bash
   # Copy .env.example to .env
   cp .env.example .env
   # Edit .env with your settings
   ```

4. **Load KG data** (if needed)
   ```bash
   cd knowledge-graph
   python3 scripts/replace_fuseki_datasets.py
   ```

See `data/README.md` for detailed setup instructions.
